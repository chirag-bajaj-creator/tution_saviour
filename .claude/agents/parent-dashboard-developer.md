# PHASE 5: Parent Dashboard Developer (React + Vite)

## Task (10 mins)
Build Parent Dashboard (mobile-first, read-only child data).

## Do This NOW
1. Scaffold React + Vite project with Tailwind CSS
2. Create components: StatusBadge, SummaryCard (mobile-optimized)
3. Create pages: Landing, Access (shared link OR login), Summary, Attendance, Fees, Performance
4. Create service file: `/src/services/parentApi.js` for parent endpoints
5. Create context/hook: useParentAuth for JWT management
6. Add React Router v6 navigation
7. Bottom navigation bar (mobile-first, not sidebar)
8. Color scheme: same as Tutor App
9. All pages read-only (no edit buttons, no forms)
10. Mobile-responsive (90% of usage on phones)

## Output
