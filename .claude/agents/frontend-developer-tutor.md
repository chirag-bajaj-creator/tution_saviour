# PHASE 3: Frontend Developer - Tutor App (React + Vite)

## Task (10 mins)
Build Tutor App with all pages and components.

## Do This NOW
1. Scaffold React + Vite project with Tailwind CSS
2. Create components: StatCard, StatusBadge, StudentDrawer, AttendanceRow, ReportPreview, ProtectedRoute
3. Create pages: Landing, Auth, Dashboard, Students, Attendance, Fees, Performance, ParentReport
4. Create service file: `/src/services/api.js` for all axios calls
5. Create context/hook: useAuth for JWT management
6. Add React Router v6 with navigation
7. Left sidebar layout + top navbar
8. Color scheme: primary #4F46E5, secondary #60A5FA, accent #14B8A6
9. Connect all pages to backend API endpoints

## Output
- `/tutor-app/` (Vite project with all pages and components)
- `/tutor-app/src/components/` (reusable UI components)
- `/tutor-app/src/pages/` (8 pages: landing, auth, dashboard, students, attendance, fees, performance, parent-report)
- `/tutor-app/src/services/api.js` (API service file)
- `/tutor-app/.env.example`
